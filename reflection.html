Reflection

ประสบการณ์จากการลงเรียนวิชานี้ Web Design Technologies

สิ่งที่ได้เรียนรู้

จากที่ได้มาเรียนวิชานี้ เพราะส่วนตัวอยากทำสาย Frontend จึงอยากฝึกการทำ Web Design ด้วย ตอนแรกคิดว่าน่าจะสนุกกับฝั่งดีไซน์ แต่พอได้ลองจริง ๆ ก็รู้ว่าตัวเองอาจจะไม่ถนัดด้าน UX/UI เท่าไร แต่ก็ถือว่าได้ลองและเข้าใจพื้นฐานของการออกแบบเว็บไซต์มากขึ้น สิ่งที่ได้เรียนรู้หลัก ๆ คือการเขียน HTML และ CSS รวมถึงการใช้ Bootstrap เพื่อช่วยจัดโครงสร้างและตกแต่งหน้าเว็บให้ดูเป็นระเบียบมากขึ้น ได้ฝึกการใช้โค้ดจริงและเห็นผลลัพธ์แบบเรียลไทม์ ซึ่งช่วยให้เข้าใจโครงสร้างของเว็บและลำดับการทำงานของโค้ดมากกว่าแค่เรียนทฤษฎี อีกอย่างคือได้ฝึกความละเอียดและความอดทนในการจัดตำแหน่ง (Layout) และปรับแต่งส่วนต่าง ๆ ให้แสดงผลถูกต้อง เพราะบางครั้งสิ่งที่เห็นบนหน้าจอไม่เหมือนที่คิดไว้ ต้องลองผิดลองถูกอยู่หลายรอบกว่าจะได้ผลลัพธ์ตามต้องการ

ปัญหาที่พบ

ไม่เคยเรียน HTML กับ CSS มาก่อน ทำให้ตอนเริ่มแรกค่อนข้างยาก เพราะต้องจำโครงสร้างแท็กและคำสั่งหลายอย่าง อีกทั้งยังไม่เคยใช้ VS Code มาก่อน ทุกอย่างใหม่หมดเลย ต้องค่อย ๆ ศึกษาและลองทำตามที่สอน

สิ่งที่อยากพัฒนาต่อ

หลังจากเรียนวิชานี้ อยากต่อยอดความรู้ไปสู่การพัฒนาเว็บไซต์แบบ Dynamic ที่มีระบบเชื่อมต่อกับฐานข้อมูล และฝึกใช้ Framework อย่าง React หรือ Vue.js ให้คล่องมากขึ้น รวมถึงฝึกออกแบบ UX/UI ให้เหมาะกับผู้ใช้จริง
